# Strategic Product Placement Analysis

**Unveiling Sales Impact with Tableau Visualization**

Full analytics pipeline: Python (Pandas) preprocessing → Tableau-ready dataset → Tableau dashboard/story → Flask app embedding the dashboard.

## Project structure

```
smartbridgeprojectmain/
├── dataset/
│   ├── Product_Positioning.csv      # Raw data
│   └── clean_product_data.csv       # Generated by preprocessing
├── data_processing/
│   └── data_preparation.py
├── flask_app/
│   └── app.py
├── templates/
│   └── dashboard.html
├── static/
│   └── style.css
├── requirements.txt
└── README.md
```

## Step 7 — Running the project

### 1. Install dependencies

```bash
cd smartbridgeprojectmain
pip install pandas flask
# or
pip install -r requirements.txt
```

### 2. Run data preprocessing

```bash
python3 data_processing/data_preparation.py
```

This loads `dataset/Product_Positioning.csv`, prints dataset info, removes duplicates, handles missing values, normalizes types and column names, adds engineered fields, and writes `dataset/clean_product_data.csv`.

### 3. Start Flask server

```bash
python3 flask_app/app.py
```

### 4. Open in browser

[http://127.0.0.1:5000](http://127.0.0.1:5000)

## Step 2 — Tableau-ready columns

The cleaned CSV supports these views:

| Visualization | Suggested fields |
|---------------|------------------|
| Average Sales Volume vs Product Category | `Product_Category` (dimension), `Sales_Volume` (measure, AVG) |
| Competitor Price vs Product Price | `Competitor_Price` vs `Price` (scatter), or `Price_vs_Competitor_Gap` |
| Product Category vs Price | `Product_Category`, `Price` (box plot or bar) |
| Consumer Demographics vs Sales Volume | `Consumer_Demographics`, `Sales_Volume` |
| Sales Volume by Product Position | `Product_Position`, `Sales_Volume` |
| Seasonal Impact on Sales | `Seasonal` or `Season_Label`, `Sales_Volume` |
| Foot Traffic vs Sales Volume | `Foot_Traffic` / `Foot_Traffic_Order`, `Sales_Volume` |

Engineered columns in `clean_product_data.csv`:

- `Competitor_Price` — renamed from competitor column
- `Price_vs_Competitor_Gap` — Price − Competitor_Price
- `Promotion_Flag` — 1/0
- `Season_Label` — Seasonal / Non-Seasonal
- `Foot_Traffic_Order` — Low=1, Medium=2, High=3 (for sorting)

## Step 8 — Tableau visualizations (how to build)

1. **Connect** to `clean_product_data.csv` (Data → New Data Source → Text file).

2. **Average Sales Volume vs Product Category**  
   - Rows: `Product_Category`  
   - Columns/Shelf: AVG(`Sales_Volume`)  
   - Chart: Bar.

3. **Competitor Price vs Product Price**  
   - Columns: `Competitor_Price`  
   - Rows: `Price`  
   - Chart: Scatter; optional detail: `Product_ID`.

4. **Product Category vs Price**  
   - Rows: `Product_Category`  
   - Columns: `Price` (median or average)  
   - Chart: Box-and-whisker or bar.

5. **Consumer Demographics vs Sales Volume**  
   - Rows: `Consumer_Demographics`  
   - Columns: SUM or AVG(`Sales_Volume`)  
   - Chart: Bar or stacked bar.

6. **Sales Volume by Product Position**  
   - Rows: `Product_Position`  
   - Columns: AVG(`Sales_Volume`)  
   - Chart: Bar.

7. **Seasonal Impact on Sales**  
   - Rows: `Season_Label` or `Seasonal`  
   - Columns: AVG(`Sales_Volume`)  
   - Chart: Bar or side-by-side.

8. **Foot Traffic vs Sales Volume**  
   - Use `Foot_Traffic` on one axis and `Sales_Volume` on the other, or bar by `Foot_Traffic` with AVG(`Sales_Volume`).

## Step 9 — Tableau dashboard with filters

1. Create a **new Dashboard**.
2. Drag sheets onto the canvas.
3. Add **filters** for:
   - **Product Category** — filter on `Product_Category`
   - **Season** — filter on `Seasonal` or `Season_Label`
   - **Consumer Demographics** — filter on `Consumer_Demographics`
4. Apply filters to all relevant sheets (use as **global** filters).

## Step 11 — Publish and embed in Flask

1. **Tableau Public**: Server → Tableau Public → Save to Tableau Public.
2. Open the workbook on the web → **Share** → **Embed Code**.
3. Copy the URL inside the `iframe` `src="..."`.
4. In `templates/dashboard.html`, set:

```html
src="YOUR_TABLEAU_PUBLIC_EMBED_URL"
```

Example format:

`https://public.tableau.com/views/YourWorkbook/YourDashboard?:showVizHome=no&:embed=y`

CSS is linked via:

```html
<link rel="stylesheet" href="{{ url_for('static', filename='style.css') }}" />
```
